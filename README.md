The system covers the following functional requirements:

### ðŸ” Super-Administrator Space

### ðŸ“Š Administrator Space (Backoffice)

## âš™ï¸ Installation & Quick Start

### Prerequisites
* Docker & Docker Compose
* Node.js 18+ (optional, for local dev without Docker)

### Quick Start
1. **Clone the repository**
   ```bash
   git clone https://github.com/ENSIAS-3A-Projects/RABAT-ENERGIE-EAU-REE.git
   cd RABAT-ENERGIE-EAU-REE
   ```

2. **Launch the Stack (Database, Backend, Frontend)**
   ```bash
   docker-compose up -d --build
   ```

3. **Access the Application**
   * **Frontend:** `http://localhost:4200`
   * **Backend API:** `http://localhost:3000`
   * **MailHog (SMTP Mock):** `http://localhost:8025`
